# Sorteo de Amigos

Pequeña app para sortear nombres aleatorios.

## Cómo usar

1. Escribe nombres en el campo
2. Añade un amigo con el botón
3. Presiona "Sortear amigo" para elegir el ganador

## Tecnologías

- JavaScript puro
- HTML5/CSS básico
